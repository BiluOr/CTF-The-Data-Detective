# CTF-The-Data-Detective

🕵️‍♂️ The Data Detective: My First Analyst CTF 📊
I am excited to share a project I've been working on lately: my very first CTF (Capture The Flag) challenge! 🎉

Inspired by the world of cybersecurity, I decided to take the CTF concept and bring it into our territory—the world of Data Investigation.

💡 Project Philosophy
This challenge isn't designed to be impossibly difficult or high-level. Instead, its mission is to make the basic process of data investigation accessible, interactive, and fun. It's about training your analytical eye rather than complex formulas.

🕹️ What’s Inside?
Step into a simulation of a Global Trading Dashboard. Buried within the data are mysteries that only a sharp analytical mind can solve by effectively using filters, cross-referencing data points, and asking the right questions.

🚀 The Challenge: 5 Levels of Investigation
The game features 5 sequential stages with increasing difficulty:

Segment Deep-Dive: Identifying top traders within complex market segments.

Growth Analysis: Analyzing YoY (Year-over-Year) trends.

Profitability Check: Spotting inconsistencies in profit margins.

Anomaly Detection: Finding outliers that don't fit the pattern.

The Grand Finale: Tracking down the "Ghost Transaction" hiding beneath the radar.

🛠️ Tech Stack
HTML5/CSS3 – Custom UI designed to mimic modern BI tools like Power BI/Tableau.

JavaScript (ES6+) – Core game logic and data filtering.

Chart.js – Interactive data visualizations.

🏁 How to Start
Clone this repository.

Open the index.html file in any modern browser.

Use the dashboard's filters and search bar to find the answers to the missions in the sidebar.

Enter the correct UUID to unlock the next level!

🏆 Are you a Data Master?
If you manage to crack all 5 levels and reach the Data Master screen, take a screenshot and share it! (But remember: no spoilers—don't reveal the answers to others! 😉).

I'd love to hear your feedback and thoughts!
